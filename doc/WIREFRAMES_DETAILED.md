## DESIGN NOTES

### Color Scheme
- Maintain current clean design
- Use accent color for "Ongoing" badges
- Different color for "Completed" badges
- Highlight collaboration CTAs

### Typography
- Clear hierarchy (H1 > H2 > H3)
- Readable body text (16px+)
- Bold for emphasis on key information

### Spacing
- Generous whitespace
- Clear section separation
- Consistent padding/margins

### Interactive Elements
- Hover states on cards
- Clear button styles
- Filter buttons with active state
- Smooth transitions

### Responsive Design
- Stack on mobile
- Maintain readability
- Touch-friendly buttons
- Collapsible sections on small screens
