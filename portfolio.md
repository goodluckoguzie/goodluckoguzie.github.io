### 1. Adaptive Prediction Horizons in Reinforcement Learning
**Status:** In Progress  
**Year:** 2024 - Present

**Description:**  
Developing an entropy-driven adaptive prediction horizon system for reinforcement learning applications. This work extends the PhD thesis research by implementing dynamic prediction horizon adjustment based on environmental complexity and action entropy.

**Key Features:**
- Entropy-driven prediction horizon selection
- Integration with SAC (Soft Actor-Critic) algorithm
- Dynamic adjustment based on environmental complexity
- Performance optimization for real-time applications

**Technologies:**
- Python, PyTorch
- Reinforcement Learning frameworks
- SocNavGym environment

**Publications:**
- Work in progress for conference submission

### 1. Predictive World Models for Social Navigation
**Status:** Completed  
**Year:** 2023

**Description:**  
Developed predictive world models that enable autonomous systems to navigate safely in social environments. Introduced 2StepAhead and Multi-Action State Predictive Models (MASPM) for improved navigation performance.

**Key Contributions:**
- 2StepAhead predictive model
- Multi-Action State Predictive Model (MASPM)
- Hybrid 2StepAhead-MASPM model
- Superior performance in dynamic social environments

### 2. Cosine-Gated LSTM Architecture
**Status:** Completed  
**Year:** 2024

**Description:**  
Introduced a novel Cosine-Gated LSTM (CGLSTM) architecture that uses cosine gating mechanisms for improved sequential learning. The architecture demonstrates superior performance in complex sequential data processing tasks.

**Key Contributions:**
- Novel cosine gating mechanism
- Dynamic input weighting based on similarity
- Superior performance over LSTM, GRU, and Transformer
- Efficient parameter usage

**Publications:**
- Published at IEEE PRML 2024 (pp. 8-15)

**Links:**
- [Paper PDF]({{ '/Cosine_Gated_LSTM_main.pdf' | relative_url }})

### 3. PhD Thesis: Social Robot Navigation
**Status:** Completed  
**Year:** 2024

**Description:**
